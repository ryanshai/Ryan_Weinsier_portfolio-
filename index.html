Engineering 11a Project 1

This project introduced students to the basics of digital fabrication and web publishing. Using a Prusa MINI+ 3D printer and Prusa Slicer software, students created a custom nametag, ran a calibration “torture test” print to explore the limits of 3D printing, and fabricated an open-source assistive device to see how digital tools support accessibility and mass customization. Students also learned to set up and host a personal portfolio website using Bootstrap and GitHub Pages, where they documented their process and reflections.

Fall 2025 · Engineering 11a

Engineering 11a Project 2

This project asks studets to remix public 3D models using Tinkercad and Autodesk Fusion, adding new features or combining designs to create novel 3D prints. Students will practice the full workflow: research and sketch ideas, model in both CAD tools, prepare in Prusa Slicer, print on a Prusa MINI+, and document each step with screenshots, photos, and notes. Finally, students will post a reflection comparing tools, processes, and outcomes.
